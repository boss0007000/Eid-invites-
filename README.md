# Eid Invites

A small animated invitation website for Eid lunch with an RSVP form that stores responses in `data/rsvps.csv`.

## Run locally

```bash
npm start
```

Then open `http://localhost:3000`.

## RSVP storage

Each RSVP submission is appended to `data/rsvps.csv`, which can be opened in Excel or similar spreadsheet apps.
